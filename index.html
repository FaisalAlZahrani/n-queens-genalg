N-Queens Problem

N-Queens problem is placing n queens on an nxn grid, such that no two queens attack each other in the same row, column, or diagonal. This visualization is considering the 8-queen problem, but the underlying source code works for any n.

Genetic Algorithm

The genetic algorithm creates a random population, assigns scores to each individual, chooses the best two from the population to create a new individual, and applies a mutation on the new individual based on the mutation rate. The procedure continues until we generate a solution, or we reach the maximum number of generations.
